# Standard Operating Procedures

## Overview

This directory contains standard operating procedures (SOPs) for common development tasks. Each command file defines the recommended approach for a specific operation.

---

## Available Commands

| Command | Purpose | Principle Alignment |
|---------|---------|---------------------|
| `constitution.md` | Constitution amendment procedure | Documented Decisions |
| `planning.md` | Project planning workflow | Incremental Delivery |
| `specification.md` | Feature specification workflow | Clarity Over Cleverness, Minimal Surface Area |
| `testing.md` | Test creation and maintenance | Testable Assertions |

---

## Usage

Refer to individual command files for detailed procedures. Each file includes:

1. Purpose statement
2. Step-by-step instructions
3. Verification checkpoints
4. Principle alignment notes
