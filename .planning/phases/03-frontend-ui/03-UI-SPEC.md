## Spacing Scale

Declared values (Vuetify defaults - multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |

Exceptions: touch targets: 44px minimum for buttons on mobile (Vuetify v-btn min-height)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF (surface) | Page background, surfaces |
| Secondary (30%) | #F5F5F5 (cards/sidebar) | Cards, navigation drawer, nav |
| Accent (10%) | #1976D2 (primary) | CTA buttons, active nav items, links |
| Destructive | #FF5252 (error) | Delete actions, destructive confirmations |

Accent reserved for: primary buttons, FAB, active navigation highlights, form submit actions, links

Default Vuetify theme palette (light):
- primary: #1976D2
- secondary: #424242
- accent: #82B1FF
- error: #FF5252
- success: #4CAF50
- warning: #FFC107
- info: #2196F3

## Responsive Breakpoints

Using Vuetify's built-in breakpoint system:

| Breakpoint | Width | Behavior |
|-----------|-------|----------|
| xs | < 600px | Mobile - single column, bottom nav option |
| sm | 600-959px | Tablet - collapsible drawer |
| md | 960-1279px | Desktop - side drawer expanded |
| lg | 1280-1919px | Large desktop |
| xl | >= 1920px | XL desktop |

Responsive utilities: `v-if="$vuetify.display.smAndUp"`, `v-if="$vuetify.display.mdAndUp"`, etc.
